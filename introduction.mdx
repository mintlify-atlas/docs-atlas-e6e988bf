## What is ArcHive?

ArcHive is a multi-platform digital capture tool designed to be your personal "second brain." In an age of information overload, ArcHive provides a calm, uncluttered, and intelligent space to effortlessly save, organize, and retrieve all the valuable pieces of information you encounter daily.

From fleeting thoughts and insightful articles to crucial code snippets, ArcHive ensures your knowledge is not just stored, but truly archived and always at your fingertips.

## Key features

### Multi-content support

Effortlessly save three types of content in one unified system:
- **Text notes** - Capture thoughts, ideas, and quick notes
- **Web links** - Save articles, documentation, and interesting pages
- **Code snippets** - Archive useful code for future reference

### Intelligent processing

ArcHive doesn't just store your content—it understands it:

<CardGroup cols={2}>
  <Card title="Automatic parsing" icon="globe">
    Automatically extracts titles, descriptions, and metadata from web links with specialized parsers for GitHub, Instagram, and YouTube
  </Card>
  <Card title="Screenshot generation" icon="camera">
    Generates visual previews for every link using Puppeteer and Cloudinary storage
  </Card>
  <Card title="Automated tagging" icon="tags">
    Intelligent tag engine analyzes content to suggest relevant tags
  </Card>
  <Card title="Platform categorization" icon="layer-group">
    Automatically categorizes links by source platform with visual badges
  </Card>
</CardGroup>

### Powerful search and organization

- **Full-text search** - Search across all content, not just titles, with keyword highlighting
- **Content type filtering** - Quickly filter by All, Link, Text, or Code
- **Platform browsing** - View all saved items from specific platforms (GitHub, YouTube, Twitter, etc.)
- **Platform statistics** - Track how many items you've saved from each source
- **Recent search history** - Quick access to previous searches
- **Infinite scroll** - Seamless pagination powered by TanStack Query

## Technology stack

ArcHive is built with modern, performant technologies:

- **Backend** - Hono framework with Bun runtime and MongoDB database
- **Mobile** - React Native with Expo for cross-platform development
- **Chrome extension** - Vanilla JavaScript with Chrome Extension APIs
- **Storage** - Cloudinary for images and screenshots
- **Search** - Full-text search with Natural language processing
- **Queue** - BullMQ with Redis for background job processing

## Philosophy

ArcHive is more than just an application—it's a testament to the belief that technology can be a profound form of art. Built with the vision to create a tool that is not only highly functional but also intuitive, aesthetically pleasing, and deeply personal.

It's about creating a digital extension of your mind - a place where information isn't just dumped, but thoughtfully curated and easily retrieved, much like a well-organized personal library.
